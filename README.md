# Pedra, Papel & Tesoura | Frontend Mentor

### Regras:

- Papel vence Pedra.
- Pedra vence Tesoura.
- Tesoura vence Papel.

Se o jogador ganha, soma 1 ponto. Se a CPU ganha, jogador perde um ponto.

Para restaurar/resetar pontuação, você pode atualizar a página.
